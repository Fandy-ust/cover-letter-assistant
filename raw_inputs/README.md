# Raw Inputs

This folder contains all user-provided source inputs for the pipeline.

## Subfolders
- `profile/`: CV, resume, LinkedIn export, and supporting profile notes.
- `job/`: raw job posts and any role-specific context notes.
- `style_samples/`: past cover letters and writing-style reference notes.

## Recommended format
Use `.md` when possible for notes you create manually. `.pdf` and `.txt` are also supported.
