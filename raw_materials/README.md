# Raw Materials

This folder contains all user-provided source inputs for the pipeline.

## Subfolders
- `personal_materials/`: CV, resume, LinkedIn export, and supporting profile notes.
- `job_description/`: raw job posts and any role-specific context notes.
- `reference_draft/`: past cover letters and writing-style reference notes.

## Recommended format
Use `.md` when possible for notes you create manually. `.pdf` and `.txt` are also supported.
